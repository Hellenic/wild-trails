# Game Sounds

This directory contains audio files for game events.

## Files

- `waypoint-found.mp3` - Sound played when a waypoint is discovered
- `goal-found.mp3` - Sound played when the goal is found

## Requirements

- File size: < 100KB each for fast loading
- Format: MP3 (widely supported)
- Tone: Positive, achievement-oriented
